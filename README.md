## Usage

<details>
  <summary><b>About EddieHub:</b> For json result about EddieHub</summary>
<br/>

```python
import eddiehub

print(eddiehub.eddiehub())
```

</details>

<details>
  <summary><b>About Member:</b> For json result about member of EddieHub</summary>
<br/>

```python
import eddiehub

print(eddiehub.member("FayasNoushad"))
```

</details>
